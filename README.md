# Pink Property — website review build

This repository is set up to publish the Pink Property website as a GitHub Pages site.

## To publish (one-time, ~2 minutes)

1. Download the review build: `pink-property-website.html` (shared separately).
2. Rename it to **index.html**.
3. In this repository on GitHub, click **Add file → Upload files**, drop in `index.html`, and **Commit changes**.
4. Wait about a minute, then open **https://prometheusdevcreator.github.io/**

The file is fully self-contained (styles, scripts and images embedded), so it also works by simply double-clicking it — no hosting required.

## To take the site down

Delete this repository (Settings → Danger Zone → Delete), and the public URL disappears with it.
